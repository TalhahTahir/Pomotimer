# Pomotimer

A simple Pomodoro timer application built with HTML, CSS, and JavaScript.

## Live Demo
The best option to use the application is the live deployment: **[https://pomotimer-rho.vercel.app/](https://pomotimer-rho.vercel.app/)**

## Files
- `index.html`: The main structure of the timer.
- `style.css`: The styling for the application.
- `script.js`: The logic for the Pomodoro timer.

## How to Run Locally

Since this is a static web application with no build steps or server dependencies, running it locally is very straightforward.

### Option 1: Open directly in a browser (Simplest)
1. Navigate to the project folder on your computer.
2. Double-click the `index.html` file.
3. The application will open in your default web browser.

### Option 2: Using VS Code Live Server (Recommended for Development)
If you are modifying the code and want to see live updates:
1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension (publisher: Ritwick Dey).
3. Open `index.html` in VS Code.
4. Right-click anywhere in the code and select **Open with Live Server**, or click the "Go Live" button in the bottom right of the VS Code status bar.
5. The application will automatically open in your default browser at `http://127.0.0.1:5500`.
